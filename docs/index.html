Finalidad:

La finalidad del juego, es que debe adelantar los coches, moviendote de derecha a izquierda y evitar la colision. Cuanto mas tiempo pase, mas rapido ira y mas probabilidad de colisionar tendra.

Instrucciones:

  • Flecha izquierda, se utiliza para movernos a la izquierda
  • Flecha derecha, se utiliza para movernos a la derecha
  • Techa de espacio, se utiliza para acelerar
